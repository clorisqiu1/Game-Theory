## 👋 简介 [⏏️](#📄-目录)
博弈论，又称为对策论（Game Theory）、赛局理论等，既是现代数学的一个新分支，也是运筹学的一个重要学科。
博弈论主要研究公式化了的激励结构间的相互作用，是研究具有斗争或竞争性质现象的数学理论和方法。
我们希望能够给出一个快速清晰的博弈论入门手册供使用者参考。
